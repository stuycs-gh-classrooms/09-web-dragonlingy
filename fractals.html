Koch Curve

The basic koch curve looks like this: 

 draw_koch(t, 4, 50)

My koch curve has the following modifications (assuming that the inputs are the ones given below):

  • If run with a depth greater than 1, every four depth 1 lines of the curve will be in the color order red, green, blue, green before repeating
  • Before each line, the turtle will leave a stamp, reminiscent of a "light" (like on a Christmas bush)
  • For every depth after depth 2, the turtle will also leave yellow stamps at the red/blue stamps rotating left & right 20 degrees, and then more stamps with a left & right rotation of the previous angle + 30 degrees for each corresponding depth.
    For example, if it is depth 4, the turtle will first leave a red stamp, rotate 20 degrees to the right, leave a yellow stamp, then rotate back to its original position, then rotate 20 degrees to the left, leave another yellow stamp, and rotate back to its original position. Then, the turtle will rotate right 50 degrees, leave a yellow stamp, etc. The depth 4 koch curve should have 4 yellow stamps for each red/blue stamp.
    These stamps are reminiscent of the glow that the "lights" give off
  • The width of the green lines drawn is doubled for each depth. For example, for depth 2 it would be width = 1, for depth 3 it would be width = 2, for depth 4 it would be width = 8, etc.
 draw_modkoch(t, 4, 50, 1, 0)

    def draw_modkoch(t, depth, length, width, lights):
      if(depth > 1):
          i = 0
          angle = 20
          t.color("red")
          t.width(1)
          t.stamp()
          while(i < lights):
              t.rt(angle)
              t.color("yellow")
              t.stamp()
              t.lt(angle)
              t.lt(angle)
              t.stamp()
              t.rt(angle)
              i += 1
              angle += 10
          t.color("red")
          draw_modkoch(t, depth-1, length, width*2, lights+1)
          t.lt(60)
          t.color("green")
          t.stamp()
          t.width(width)
          draw_modkoch(t, depth-1, length, width*2, lights+1)
          t.rt(120)
          i = 0
          angle = 20
          t.color("blue")
          t.width(1)
          t.stamp()
          while(i < lights):
              t.rt(angle)
              t.color("yellow")
              t.stamp()
              t.lt(angle)
              t.lt(angle)
              t.stamp()
              t.rt(angle)
              i += 1
              angle += 30
          t.color("blue")
          draw_modkoch(t, depth-1, length, width*2, lights+1)
          t.lt(60)
          t.color("green")
          t.stamp()
          t.width(width)
          draw_modkoch(t, depth-1, length, width*2, lights+1)
      elif(depth == 1):
          t.fd(length)

Sierpinski Triangle

The basic sierpinski triangle looks like this: 

 draw_sierpinski(t, 4, 500)

My sierpinski triangle has the following modifications (assuming that the inputs are the ones given below):

  • For every depth 1 line drawn/every line that composes the triangle, there is a 1/5 or 20% chance that the line will "not be drawn" (the turtle will draw a white line so it isn't visible)
  • For every depth, a random value between 0-99 is divided by 255 (to convert to floating points for the rgb values) is added to the parameters redc, greenc, and bluec. These parameters are then subtracted from 1 and used as the turtle's corresponding rgb values. (If any accumulative sum [redc, greenc, bluec] becomes greater than 1, it will be reset to 0 to avoid errors).
    Thus, the colors used to draw the sierpinski triangles generally become darker as depth increases
  • Note: if depth is inputted as 1 and the original redc, greenc, bluec colors are inputted as 0, the triangle will be white. 
 draw_modsierpinski(t, 4, 500, 0, 0, 0)

    def draw_modsierpinski(t, depth, length, redc, greenc, bluec):
        t.color(1 - redc, 1 - greenc, 1 - bluec)
        if(depth > 1):
            redc += randto(100) / 255.
            bluec += randto(100) / 255.
            greenc += randto(100) / 255.
            if(redc > 1):
                redc = 0
            if(bluec > 1):
                bluec = 0
            if(greenc > 1):
                greenc = 0
            draw_modsierpinski(t, depth - 1, length / 2, redc, greenc, bluec)
            t.fd(length/2)
            draw_modsierpinski(t, depth - 1, length / 2, redc, greenc, bluec)
            t.lt(120)
            t.fd(length/2)
            t.rt(120)
            draw_modsierpinski(t, depth - 1, length / 2, redc, greenc, bluec)
            t.rt(120)
            t.fd(length/2)
            t.lt(120)
        elif(depth == 1):
            if(randto(5) == 0):
                t.color("white")
                triangle(t, length)
                t.color(1 - redc, 1 - greenc, 1 - bluec)
            else:
                triangle(t, length)

Tree

The basic tree looks like this: 

 tree(t, 6, 50, 30)

My tree has the following modifications (assuming that the inputs are the ones given below):

  • For each depth, before the first branch is drawn, an amount is added to the rotation of the angle. The amount was originally given as 0 (according to inputs). There is a 1/3 chance the amount will increase by 10 and a 2/3 chance it will decrease by 5. That amount is added to the angle rotated by the turtle when drawing branches. Changes to the ammount are accumulative.
  • For each depth, before the first and second branches are drawn, there is a 1/3 chance the turtle will become red and 2/3 it will become green. The first branch "roll" determines both the changes to amount as well as the color of the turtle.
  • For depth 1, the turtle will be randomly set to a width between 1 and 10 before drawing. 
 modtree(t, 6, 50, 30, 0)

    def modtree(t, depth, length, angle, amount):
      if(depth > 1):
          t.fd(length)
          if(randto(3) == 0):
              amount += 10
              t.color("red")
          else:
              amount -= 5
              t.color("green")
          t.rt(angle + amount)
          modtree(t, depth - 1, length, angle, amount)
          t.lt((angle+amount)*2)
          if(randto(3) == 0):
              t.color("red")
          else:
              t.color("green")
          modtree(t, depth - 1, length, angle, amount)
          t.rt(angle + amount)
          t.bk(length)
      else:
          t.width(random.randrange(1, 11))
          t.fd(length)
          t.bk(length)
